# LSESU Economics Practice Platform

Static GitHub Pages practice site for the LSESU Economics Society Economics Challenge.

## Contents

- 105 questions imported from `LSE经济学竞赛.pdf`
- 4 practice sections: Macro, Micro, Quantitative, Sample Test
- 105 answers and explanations
- 20 diagram/table questions with cropped source images

## Local Preview

Open `index.html` directly, or run a static server in this folder.

```bash
python3 -m http.server 8784
```
